This assignment took me quite a while to work through and was fairly challenging to me because I have not actually used HTML or CSS except for the small examples in class. Originally, I had the CSS and HTML files separate because that seemed to be the easiest way to do it. However, once I wanted to actually deploy the code, it would not use the CSS file how I wanted it to. So, I decided to have the CSS within the HTML and use the style tags to do so.

First, I designed the grid area and adjusted the colors and sizes so they fit the proportions outlined in the assignment. Then, the first area I started with was the area 2 with my name and fake phone number, but I did not find the phone icon until I got to area 8--where the other icons are. Then, I put my image in the area 1, but this was a struggle for a while because the photo was either too large or stretched the grid incorrectly. Once I remembered the height of the grid being 100px, I adjusted the picture to fix and stopped adjusting the width at all. For the this third area, I left for last so I could explain how I created the site. Area 4 was not too difficult once I found some instructions on how to build tables online, but adjusting the format was more difficult. Then, for area 5, I added a photo of machine code because of the CIS 351 class I took last semester. Area 6, I input a link to GVSU and Applied Innovation, where I am currently interning at. In area 7, I did not know what to put for multiple paragraphs, so I pasted in an exerpt from Harry Potter and the Sorcerer's Stone. Finally, for Area 8 with the icons was a little difficult to figure out for a bit, until I found a website called fontawesome that had very easy ways to use the icons I needed. I then adjusted them using Flexbox.

Overall, I definiely learned a lot about HTML and CSS and enjoyed this challenge. I think the most difficult part was formatting at times, but I found many useful resources online as well as used the lecture slides to create this.
